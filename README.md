# To-Do List — DOM Manipulation (Lab 9)

## Features
✔ Add new tasks  
✔ Edit tasks  
✔ Delete tasks  
✔ Mark tasks as completed  
✔ Save tasks using localStorage  
✔ Auto-load tasks on page refresh

## How to Run
1. Open the folder in VS Code.
2. Right-click index.html.
3. Select **Open with Live Server**.
